
# Task List API Laravel 9

Manage Task list with API.




## Requirement

**REQ:** PHP 8.1


## Installation

Install my-project

```bash
composer install
```
Update .env file with your DB connection 
and mail so email can be tigger

For Migration
```base
php artisan migrate --seed
```
